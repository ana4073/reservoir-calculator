# Калькулятор РВС/РГС - Расчет кубатуры резервуаров

Полнофункциональное десктопное приложение на C# для расчета объема и параметров резервуаров вертикальных стальных (РВС) и резервуаров горизонтальных стальных (РГС) с 3D визуализацией.

## Возможности

### Расчеты РВС (Резервуары вертикальные стальные)
- Расчет объема цилиндрического резервуара
- Поддержка различных типов крыши:
  - Коническая крыша
  - Плоская крыша
  - Полусферичная крыша
- Расчет массы материала на основе толщины стенок
- Расчет площади поверхности

### Расчеты РГС (Резервуары горизонтальные стальные)
- Расчет объема горизонтального цилиндра
- Поддержка различных типов днища:
  - Плоское днище
  - Эллиптическое днище
  - Полусферичное днище
- Расчет массы материала
- Расчет площади поверхности

### 3D Визуализация
- Интерактивная 3D модель резервуара
- Поворот, масштабирование и перемещение модели
- Обновление модели в реальном времени при изменении параметров
- Детальная визуализация всех компонентов (стенки, днище, крыша/дно)

### Функции приложения
- Параметрические ползунки для удобного изменения размеров
- Ввод значений вручную с валидацией
- Расчет и отображение результатов в реальном времени
- Сброс параметров к значениям по умолчанию
- Экспорт результатов в текстовый файл

## Технические характеристики

- **Платформа**: Windows (WPF - Windows Presentation Foundation)
- **Язык**: C# (.NET 6.0)
- **Визуализация**: HelixToolkit.Wpf (3D графика)
- **Архитектура**: MVVM (Model-View-ViewModel)

## Установка

### Требования
- .NET 6.0 Runtime или выше
- Windows 7 и выше

### Построение из исходного кода

```bash
# Клонирование репозитория
git clone https://github.com/ana4073/reservoir-calculator.git
cd reservoir-calculator

# Восстановление пакетов NuGet
dotnet restore

# Построение проекта
dotnet build

# Запуск приложения
dotnet run --project ReservoirCalculator/ReservoirCalculator.csproj
```

### Компиляция в EXE файл

**Self-contained версия (работает без .NET Runtime):**
```bash
dotnet publish -c Release -r win-x64 --self-contained
```

**Компактная версия (требует .NET Runtime):**
```bash
dotnet publish -c Release -r win-x64
```

Exe файл будет находиться в: `ReservoirCalculator/bin/Release/net6.0-windows/win-x64/publish/ReservoirCalculator.exe`

## Использование

1. **Выберите тип резервуара**: РВС или РГС
2. **Установите основные параметры**:
   - Диаметр (м)
   - Высота или Длина (м)
   - Толщина стенок (мм)
3. **Для РВС выберите**:
   - Тип крыши (Коническая, Плоская, Полусферичная)
   - Высоту крыши (м)
4. **Для РГС выберите**:
   - Тип днища (Плоское, Эллиптическое, Полусферичное)
5. **Просмотрите результаты**:
   - Объем в м³
   - Массу материала в кг
   - Площадь поверхности в м²
6. **Используйте 3D визуализацию**:
   - Вращайте модель мышью
   - Масштабируйте колесом мыши
   - Перемещайте зажав среднюю кнопку мыши
7. **Экспортируйте результаты** в текстовый файл

## Математические формулы

### РВС (Вертикальный резервуар)
- **Объем цилиндра**: V = π × r² × h
- **Объем конической крыши**: V = (1/3) × π × r² × h
- **Объем полусферичной крыши**: V = (2/3) × π × r³

### РГС (Горизонтальный резервуар)
- **Объем цилиндра**: V = π × r² × L
- **Объем эллиптического днища**: V = (1/6) × π × D × h (для двух дниш)
- **Объем полусферичного днища**: V = (4/3) × π × r³ (для двух дниш)

### Масса материала
- **Объем материала**: V_материала = Площадь_поверхности × Толщина
- **Масса**: M = V_материала × Плотность_стали (7850 кг/м³)

## Примеры использования

### Пример 1: РВС с конической крышей
- Диаметр: 5 м
- Высота: 8 м
- Толщина стенок: 10 мм
- Результат: Объем ≈ 157 м³, Масса ≈ 3800 кг

### Пример 2: РГС с эллиптическим днищем
- Диаметр: 4 м
- Длина: 10 м
- Толщина стенок: 8 мм
- Результат: Объем ≈ 126 м³, Масса ≈ 2900 кг
