# Reservoir Calculator Pro - ПРОФЕССИОНАЛЬНАЯ ВЕРСИЯ 🚀

## 🎯 О проекте

ПОЛНОЦЕННОЕ приложение для расчета, проектирования и калькуляции резервуаров:
- **РВС** (Резервуары Вертикальные Стальные)
- **РГС** (Резервуары Горизонтальные Стальные)

## ⚡ Основные возможности

### 1️⃣ Расчет резервуаров
- Точный расчет объема и массы
- Поддержка различных типов крыш и днищ
- Расчет по ГОСТ стандартам

### 2️⃣ Каталог комплектующих (19+ позиций)
- **Люки** (верхние, нижние, прямоугольные)
- **Патрубки** (всасывающие, напорные, переливные)
- **Лестницы** (внутренние, внешние, трапы)
- **Клапаны** (обратные, отсечные, предохранительные)
- **Фланцы** (различные диаметры)
- **Крепёж** (болты, гайки)
- **Уплотнения** (резиновые прокладки)
- **Изоляция** (теплоизоляционные материалы)
- **Покрытия** (краски, эмали)

### 3️⃣ Конструктор комплектующих
- Выбор нужных деталей из каталога
- Стандартные наборы для резервуаров
- Расчет общего веса и стоимости
- Управление количеством компонентов

### 4️⃣ Материалы и расценки
**5 видов материалов:**
- Сталь Ст3сп (45 руб/кг)
- Сталь 09Г2С (52 руб/кг)
- Нержавеющая сталь AISI 304 (280 руб/кг)
- Нержавеющая сталь AISI 316 (350 руб/кг)
- Алюминий АМГ-3 (180 руб/кг)

### 5️⃣ Расчет затрат
- Стоимость материала резервуара
- Стоимость комплектующих
- Общая сметная стоимость
- Детализированная калькуляция

### 6️⃣ Спецификации и сметы
- Автоматическая генерация спецификации
- Экспорт в текстовый формат
- Подготовка к экспорту в PDF/Excel
- Форматированные отчеты

### 7️⃣ 3D визуализация
- Интерактивная 3D модель резервуара
- Отображение компонентов
- Детальная визуализация конструкции

## 📊 Технические характеристики

- **Платформа**: Windows (WPF)
- **Язык**: C# (.NET 6.0)
- **3D графика**: HelixToolkit.Wpf
- **Архитектура**: MVVM
- **База данных**: Встроенные коллекции (можно расширить до SQL)

## 🛠️ Установка и запуск

### Требования
- Windows 7+
- .NET 6.0 Runtime
- Visual Studio 2022 (для разработки)

### Запуск
```bash
cd reservoir-calculator
dotnet restore
dotnet run --project ReservoirCalculator/ReservoirCalculator.csproj
```

### Компиляция в EXE
```bash
dotnet publish -c Release -r win-x64 --self-contained
```
